# 🌐 Interactive Responsive Webpage with API Integration

## 📌 Project Overview

This project is an advanced interactive webpage built using **HTML, CSS, and JavaScript**. It not only includes responsive design and user interaction but also integrates an external API to fetch and display dynamic data from GitHub.

---

## ✨ Features

* Clean and structured HTML layout
* External CSS for styling and responsiveness
* JavaScript for interactivity
* Button to dynamically change background color
* External profile link (LinkedIn)
* API integration to fetch real-time GitHub data
* Dynamic content rendering on the webpage

---

## 🛠️ Technologies Used

* **HTML5** – webpage structure
* **CSS3** – styling and responsive design
* **JavaScript (ES6)** – interactivity and API handling
* **GitHub API** – to fetch user profile data

---

## 📂 Project Structure

```id="m1t8pf"
project-folder/
│
├── index.html     # Main HTML file
├── styles.css     # Styling file
└── script.js      # JavaScript file (logic + API integration)
```

---

## 🚀 How to Run the Project

1. Create a folder on your system
2. Add the following files:

   * `index.html`
   * `styles.css`
   * `script.js`
3. Ensure all files are in the same directory
4. Open `index.html` in any web browser

---

## ⚡ Functionality

### 🔹 Background Color Change

* Clicking the **“Change Background Color”** button changes the background color randomly.

### 🔹 External Profile Link

* The **“Visit My Profile”** button opens a LinkedIn profile in a new tab.

### 🔹 GitHub API Integration

* Fetches data from GitHub API:

  * Name
  * Bio
  * Number of public repositories
  * Followers
* Displays the data dynamically on the webpage
* Handles errors if API fails

---

## 📱 Responsive Design

The webpage adapts to different screen sizes:

* **Desktop** → Full layout
* **Tablet (≤768px)** → Adjusted fonts and layout
* **Mobile (≤480px)** → Optimized for smaller screens
